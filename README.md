Computer Vision Subsystem for Autonomous Robotic Bin Picking

a computer vision subsystem that enables a robotic arm to see into a bin of densely packed industrial
parts, identify each object, and determine exactly where and how it is oriented—all with enough speed
and precision for the robot to pick up each part reliably without human intervention. The system should
achieve ≤ 5 mm localization accuracy at the grasp plane with a pick success rate of ≥ 80% (stretch goal: ≥
90%), operating at cycle times under 4 seconds per pick.
The long-term ambition is a modular, repeatable perception pipeline that can be retrained for new part types
in hours rather than weeks—making it practical for small-batch manufacturing environments like those of
Indiana Tech's regional industry partners.
